# MiddleOrbit Brand System

- **Palette**
  - Midnight Navy `#0B1B3A`
  - Orbit Teal `#1ED6C1`
  - Ion Orange `#FF7A18`
  - Cloud Silver `#EEF1F5`
  - Nebula Indigo `#4F56FF`
- **Typography**
  - Display: Space Grotesk (600/700)
  - Body: Inter (400/500)
- **Imagery**
  - Soft gradient orbits, subtle starfields, rounded cards with frosted glass.
- **Voice**
  - Confident, transparent, data-backed. Emphasize trust, automation, ROI.
- **Components**
  - Rounded-3xl cards, soft shadows (`shadow-glow`), micro-interaction hover lifts.
- **Iconography**
  - Duotone icons representing AI, maps, contracts, analytics.
