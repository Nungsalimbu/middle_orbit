# Roadmap

## MVP (0-3 months)
- Auth (OAuth + JWT), 2FA scaffolding.
- Employer + Candidate onboarding flows.
- Job posting, applications, AI Career-Fit v1 (rule-based).
- Dashboards with ROI metrics + placement timeline basic view.
- Blog, case studies, SEO baseline with JSON-LD.

## v1 (3-6 months)
- Interactive talent map with geospatial heatmaps.
- Placement transparency timeline sharing + notifications.
- Micro-training modules and badge issuance.
- Referral engine with payout tracker.
- Contract generator + e-sign integration.
- Talent subscription marketplace billing.

## v2 (6-12 months)
- AI interview simulator with voice/video playback.
- Advanced analytics (Mixpanel + Looker dashboards).
- Multi-region landing pages with localization.
- Automated pen-test + compliance reporting.
- Marketplace monetization (usage-based billing), SLA automation.
